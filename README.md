# Simulação de Controlador Fuzzy para Controle Ativo de Vibração (CAV) em Viga Engastada

Este repositório contém o código de simulação computacional e modelagem matemática para um sistema de **Controle Ativo de Vibração (CAV)** de uma viga flexível engastada-livre.

O principal objetivo do projeto é validar, via simulação em ambiente MATLAB, a eficiência de um controlador baseado em lógica Fuzzy para atenuar vibrações estruturais induzidas por uma perturbação harmônica externa (motor desbalanceado).
